# eyesim-docker
Docker images for EyeSim 

## How to Use
Make sure you have a docker  
```
$ docker build -t eyesim1.5.2:alpha-full-0.0.1 . 
$ docker compose up
```

![eyesim running in a docker container](eyesim-docker.png)

## Support Version
```ubuntu 20.04``` ```EyeSim version 1.5.2```
